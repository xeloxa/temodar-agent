What WP-Hunter does

WP-Hunter is a local reconnaissance and static analysis tool for WordPress plugins/themes. It helps you discover targets, prioritize risk with metadata signals, and run deep Semgrep-backed checks.

  • Discover plugin/theme targets from WordPress directory pages.
  • Prioritize by installs, update freshness, and risk score signals.
  • Run Semgrep scans and review issue/clean status quickly.
